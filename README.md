# Finance Tracker

A full-stack personal finance management application that helps users track income, expenses, budgets, and financial insights.

## Features

- Add, edit, and delete transactions
- Track income and expenses
- Categorize spending
- Monthly financial summaries
- Interactive charts and analytics
- User authentication
- Responsive design
- Export data to CSV

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Chart.js

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/finance-tracker.git
cd finance-tracker
npm install
npm start
```
